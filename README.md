# Speech Converter

# Description
- This project lets users convert text into speech.
- Users can write a message in the textbox and have it read out loud.

# Features
- Convert written text into speech
- Select different voices
- Speak button
- Theme toggle

# Tools
- HTML
- CSS
- JavaScript
